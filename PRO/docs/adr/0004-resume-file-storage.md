# ADR-0004: 简历文件存储方案

**日期**: 2026-05-21
**状态**: 已接受

## 背景

学生投递岗位需上传 PDF 简历。生产环境应使用云存储（OSS），但开发/实训环境没有 OSS 服务。

## 决策

- 开发期：存入后端本地文件夹（`./uploads/resumes`）
- 文件命名：`{timestamp}_{uuid}.pdf`（避免文件名冲突）
- 格式限制：仅 `.pdf`，最大 5MB
- 架构预留文件存储抽象层（`FileStorageService` 接口），生产期实现 `OssFileStorageService` 替换

## 后果

- **正面**：开发零依赖，无需配置 OSS；接口抽象使切换成本低
- **负面**：本地存储无冗余，服务器迁移需拷贝文件
- **风险**：并发写入冲突（文件名随机化避免）、磁盘空间不足
